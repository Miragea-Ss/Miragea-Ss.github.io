# Submission Checklist

Safety deadline: 2026-07-30. Official deadline: 2026-08-03 17:00 EDT / 2026-08-04 06:00 JST.

## Product evidence

- [x] Public app exists.
- [x] Browser-side SHA-256 verification works.
- [x] Real Genblaze Pipeline source exists.
- [x] ComfyUI API and existing-file fallback exist.
- [x] Offline tests pass.
- [ ] At least three representative real assets are in the catalog.
- [ ] Credentialed B2 run has durable asset and manifest URLs.
- [x] `b2-proof.json` is generated from a real B2 run.

## Public delivery

- [x] Astro production build succeeds locally.
- [x] Built page loads without console errors.
- [x] Current embedded video loads and plays.
- [ ] Source changes are committed and pushed with user approval.
- [ ] GitHub Pages deployment completes.
- [ ] Public page passes logged-out desktop and mobile checks.
- [ ] Source link resolves publicly.

## Final video

- [ ] Record after B2 proof and final UI are live.
- [ ] 1920x1080, English, under three minutes.
- [ ] Show the working app, source, B2 manifest URI, failure fallback, and browser verification.
- [ ] Upload publicly and test playback while logged out.

## Devpost

- [ ] Working app URL entered.
- [ ] GitHub repository URL entered.
- [ ] Providers and models listed exactly.
- [ ] B2 and Genblaze usage explained.
- [ ] Final demo video URL entered.
- [ ] Screenshots added.
- [ ] Submission preview checked while logged out.
- [ ] Final submission completed by 2026-07-30.

## Optional prize leverage

- [ ] Prepare one substantive Genblaze issue or feature request.
- [ ] Submit it only after user approval.
